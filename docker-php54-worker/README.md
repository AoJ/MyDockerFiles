# Docker php54 worker

- php 5.4 with node.js and some dev dependencies

## Usage

### in Dockerfile

    FROM aooj/php54-worker:latest

## Contributing

### buil
    git clone https://github.com/AoJ/docker-php54-worker.git
    make build
    
### and start it
    make run

### or build, start and attach
    make debug
